# NimbleBlocks

A fast, turn-based Xamarin game inspired by Nim.  
Players remove blocks from one row per turn, and the player who takes the last block wins.

## Built With

- Xamarin.Forms
- Xamarin.Essentials
- SkiaSharp
- .NET Standard shared game logic

## Features

- Player vs AI and Player vs Player modes
- Difficulty-based timers (Easy, Intermediate, Difficult)
- Avatar/name setup for players
- Animated game board and UI effects
- Sound effects and background music controls

## Run Locally

1. Open `NimbleBlocks.sln` in Visual Studio.
2. Restore NuGet packages.
3. Set either `NimbleBlocks.Android` or `NimbleBlocks.iOS` as startup project.
4. Build and run on an emulator or physical device.
